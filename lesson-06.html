NodeJS

  • Softwarový systém navrhnutý pre škálovateľné internetové aplikácie, predovšetkým webových serverov.
  • Obsahuje V8 JavaScript engine od spoločnosti Google a niekoľko štandardných knižníc na prácu so súbormi a sieťou
  • npm - správca balíčkov

Grunt

  • JavaScript task runner
  • automatizácia, uľahčenie a zrýchlenie práce
  • my použijeme modul browserify spúšťaný gruntom na vyskladanie výsledného JS súboru

JSON

  • najbežnejší formát na prenos dát

{
    "id": 101,
    "date": null,
    "name": "Anakin Skywalker",
    "arr": ["anakin@tatooine.com", "Tattoine"],
    "obj": {
        "species": "human",
        "force": "light"
    }
}

XML

  • ďalší bežný formát na prenos dát

<item id="101">
    <date></date>
    <name>Anakin Skywalker</name>
    <arr>
        <arr-item>anakin@tatooine.com</arr-item>
        <arr-item>Tattoine</arr-item>
    </arr>
    <obj>
        <property name="species">human</property>
        <property name="force">light</property>
    </obj>
</item>
